# Privacy Policy — Robotics Tool

Last updated: 2026-03-22

Robotics Tool does not collect, store, or share any personal data.

## Data We Do Not Collect
- Personal information
- Location data
- Usage analytics
- Any data transmitted through ROS2 topics

## Local Storage
The app stores only:
- Last connected IP address (stored locally on device only)
- Theme preference (stored locally on device only)
